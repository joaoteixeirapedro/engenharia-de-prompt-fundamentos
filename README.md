# engenharia-de-prompt-fundamentos
Prompt genérico : Oque e LLM ?
LLM significa “Large Language Model” (Modelo de Linguagem de Grande Escala).
Basicamente, é um tipo de inteligência artificial treinada com MUITO texto (livros, sites, artigos, etc.) para conseguir:
entender o que você escreve
responder perguntas
traduzir idiomas
escrever textos (histórias, redações, código, etc.)
conversar de forma natural
💡 Em termos simples:
Um LLM é um modelo que aprende padrões da linguagem humana e usa isso pra prever qual palavra vem depois — e assim formar respostas coerentes.
🧠 Exemplos de LLMs:
ChatGPT
GPT-4
Claude
Gemini
⚙️ Como funciona (bem resumido):
Ele não “pensa” como humano — ele calcula probabilidades de palavras com base no que aprendeu durante o treinamento.

Prompt estruturado : Você é um professor experiente e paciente, especializado em lógica de programação. Explique o que é “LLM” (Large Language Model) de forma simples. Use linguagem acessível, limite a resposta a três parágrafos e responda em português brasileiro. Forneça também um exemplo.

Um LLM (Large Language Model) é um tipo de inteligência artificial treinada para entender e gerar texto parecido com o de humanos. Ele aprende analisando uma enorme quantidade de textos (livros, sites, artigos) e, com isso, consegue prever qual palavra vem depois de outra, formando frases coerentes. Pense nele como um “autocompletar super avançado” que não só completa frases, mas também responde perguntas, explica conceitos e até escreve histórias.

Na prática, ele não “pensa” como uma pessoa nem tem consciência. Ele funciona identificando padrões na linguagem. Quando você faz uma pergunta, o modelo analisa o que você escreveu e gera uma resposta baseada no que aprendeu durante o treinamento. Por isso, ele consegue conversar, traduzir, resumir textos e ajudar em programação.

Exemplo:
Você pergunta: “O que é uma variável em programação?”
O LLM pode responder: “Uma variável é um espaço na memória do computador usado para armazenar um valor que pode mudar ao longo do tempo, como um número ou texto.”
